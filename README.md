# Requirements

Install [pnpm](https://pnpm.io/).

We recommend an LTS version of [Node.js](https://nodejs.org/en/) >= 16.

# Local Development

To work on Rxdrag web apps, from the root,

1. Run `pnpm install`

2. Run `pnpm dev`
